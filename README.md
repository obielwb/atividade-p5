# Entrega: Pinturas com Código

**Referência da pintura:** 
"(Concentric Squares. Max Bill, 1969)"

**Imagem da pintura:**
[Link para a pintura no Artsy](https://www.artsy.net/artwork/max-bill-concentric-squares)

**Arquivos:**
- `index.html`
- `GabrielWillianBartmanovicz.js`

**Descrição da animação ou interação:**
A interação ocorre através do clique do mouse no canvas. Sempre que o usuário clica na tela, toda a estrutura geométrica das espirais realiza uma rotação de 90° (graus). A animação da rotação é feita de forma suave e gradual utilizando a função matemática de interpolação `lerp()`, que garante uma transição visualmente fluida entre a rotação atual e a nova rotação alvo.
